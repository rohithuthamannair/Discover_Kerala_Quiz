# Discover Kerala – Harmony Day Cultural Quiz Game

## Project Overview

Discover Kerala is an educational Pygame application designed to introduce primary school students to selected aspects of Kerala culture through interactive learning screens and a quiz-based assessment.

Students can explore:

* Food
* Festivals
* Arts and Performance
* Traditional Clothing
* Martial Arts
* Wellness Practices

After learning about Kerala culture, students complete a multiple-choice quiz to test their understanding.

---

## Educational Purpose

The game was designed as a digital cultural-learning activity that can be deployed on classroom computers or tablets during Harmony Day and multicultural school events.

### Example Scenario

A teacher sets up multiple cultural learning stations:

* Table 1 – Chinese Culture Activity
* Table 2 – Indigenous Australian Culture Activity
* Table 3 – Discover Kerala Game
* Table 4 – Italian Culture Activity
* Table 5 – Japanese Culture Activity

Students rotate between stations and independently complete each activity.

### At the Kerala Station

Students:

1. Enter their name
2. Explore cultural information
3. Complete the quiz
4. Receive a final score

This encourages cultural awareness and supports the goals of Harmony Day.

---

## Features

* Start screen
* Student name entry
* Interactive cultural learning screens
* Five-question multiple-choice quiz
* Automatic score calculation
* Results stored in a text file
* Support for multiple students
* Pydoc-generated documentation

---

## File Handling

The program stores student results in:

```text
student_results.txt
```

Each completed quiz is saved with:

* Student number
* Student name
* Quiz score

### Example

```text
1. Rohith - 5/5
2. Joel - 4/5
3. Meng - 5/5
```

This allows teachers to:

* Track participation
* View student scores
* Review learning outcomes after the activity

## How to Run

### 1. Open the project folder

Open the project folder in VS Code or your preferred Python IDE.

### 2. Install Pygame

```bash
pip install pygame
```

### 3. Run the game

```bash
py -3.12 main.py
```
