Discover Kerala – Harmony Day Cultural Quiz Game

Project Overview
The game introduces students to selected aspects of Kerala culture through interactive learning screens and a quiz-based assessment

Students can explore:

Food
Festivals
Arts and Performance
Traditional Clothing
Martial Arts
Wellness Practices

After learning about Kerala culture, students complete a multiple-choice quiz to test their understanding.

Educational Purpose

The game was designed as a digital cultural-learning activity that can be deployed on classroom computers or tablets during Harmony Day and multicultural school events.

For example:

Table 1 – Chinese Culture Activity
Table 2 – Indigenous Australian Culture Activity
Table 3 – Discover Kerala Game
Table 4 – Italian Culture Activity
Table 5 – Japanese Culture Activity

Students rotate between stations and independently complete each activity.

At the Kerala station students:

Enter their name.
Explore cultural information.
Complete the quiz.
Receive a final score.

This encourages cultural awareness and supports the goals of Harmony Day.

Features
Start screen
Student name entry
Interactive cultural learning screens
Five-question multiple-choice quiz
Automatic score calculation
Results stored in a text file
Support for multiple students
Pydoc-generated documentation
